## 下一步需要在Unity编辑器中完成

### 1. 创建场景
需要在Unity编辑器中创建以下场景：

**Launcher场景：**
1. 创建空GameObject命名为"GameManager"，挂载 GameManager.cs
2. 这个场景会自动初始化所有管理器并检查热更新

**Login场景：**
1. 创建Canvas和EventSystem
2. 创建三个Panel：LoginPanel, RegisterPanel, ForgotPasswordPanel
3. 为每个Panel添加对应的UI元素（InputField, Button, Text等）
4. 挂载对应的UI脚本（LoginUI, RegisterUI, ForgotPasswordUI）

### 2. 配置Build Settings
在Unity菜单 File → Build Settings 中添加场景：
1. Launcher.unity
2. Login.unity
3. SampleScene.unity

### 3. 安装Addressables包（可选）
如果需要完整的热更新功能：
1. Window → Package Manager
2. 搜索"Addressables"并安装
3. Window → Asset Management → Addressables → Groups
4. 配置远程资源路径

### 4. 配置服务器地址
修改 [ServerConfig.json](e:/unity project/FPS/Assets/Resources/Config/ServerConfig.json)：
```json
{
    "serverUrl": "你的服务器地址",
    "cdnUrl": "你的CDN地址",
    "timeout": 30
}
```

## 测试流程

1. 运行Launcher场景
2. 观察Console日志，查看初始化和热更新检查过程
3. 进入Login场景后测试：
   - 游客登录
   - 账号注册
   - 账号登录
   - 记住密码
   - 找回密码

## 注意事项

- 目前服务器API是模拟的，需要后端开发者实现对应接口
- 热更新功能需要配置Addressables才能完整使用
- 所有密码都经过SHA256哈希后传输
- Token使用AES加密存储在本地
